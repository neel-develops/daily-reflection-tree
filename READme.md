# Daily Reflection Tree

A deterministic reflection system based on structured decision trees.

## Features
- No AI at runtime
- Fully deterministic branching
- Covers 3 psychological axes:
  - Locus (Control)
  - Contribution
  - Perspective

## Structure
/tree → contains tree JSON
/agent → optional CLI runner

## How to Run (Optional Agent)
python main.py

## Goal
Help employees reflect through structured thinking instead of free-form AI.
